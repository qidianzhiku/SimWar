# R7 BFF Endpoint Boundary

This package is an internal contract draft. It is not an endpoint and does
not authorize implementation, deployment, Teacher UI, Student UI, scenario
runtime activation, PostgreSQL, Pilot, or Production.

The future endpoint must receive explicit tenant and scenario context and
must return a role-appropriate projection. The Student projection remains
redacted. The Teacher projection cannot write formal truth. Any implementation
requires a separate owner-authorized gate, route-level authorization tests,
tenant and visibility tests, and an independent evidence review.
